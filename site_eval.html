SlideWiki Accesibility

SlideWiki accesibilty have been evaluated using Web2Access Framework, which is based on WCAG 2.0 Guidlines. The testing results can be seen in the table below.

Test # Test Name Test Outcome
1 Login, Signup and Other Forms 33%
2 Image ALT Attributes and Text Alternative 67%
3 Link Target Definitions 67%
4 Logical Page Structure and Order 100%
5 Removal of Stylesheet 67%
6 Audio/Video Features 100%
7 Video/animations - audio descriptions 100%
8 Appropriate use of Tables 100%
9 Tab Orderings Correct and Logical 33%
10 Page Functionality with Keyboard and Touch Screen 0%
11 Accessibility of Text Editors 33%
12 Appropriate Feedback 67%
13 Contrast and Colour Check 33%
14 Page Integrity when Zooming 100%
15 Text size, style, blinking elements and Readability 67%

Even though many of tests achieve high results, it can be concluded that SlideWiki is an inaccessible web application, due to some of itâ€™s critical functionality such slide editing being inaccessible. This web application has not been developed with accessibility in mind.

SlideWiki Usability

SlideWiki usability has been evaluated using UserFocus Web usability guidelines. It can be concluded that application lacks attractive and clear homepage and some basics tips on how to use the mainfunctionality, which can easily put off new users. Plus, access to the content generated by other usersis vague.
