Introduction

Accesibile software design refers to creating sofware which is ussable by a person with a dissability or impairement. Quite often accesing such software requires using assistive technologies such as Screen Reader, Magnifier or Braille device.

This website presents accesibility and usuability evalution results for two Web 2.0 applications: SlideWiki and Gmail, and for two desktop applciations: Skype and Notepad++.

An accesible, web based oral presentation has been created, with a synchonised transcript. It is available under the follow link: Web based oral presentation on SlideWiki.

This website has been desinged to accesible itself. It has been evaluated for it's accesibility, usability, design to standards and cross compability. This evaluation can be founde under the following link: Evaluation of this Website.
